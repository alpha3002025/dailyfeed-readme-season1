# Dailyfeed 소개
아직은 프로젝트 문서를 작성중입니다. 10월 마지막주에 모두 완료될 예정입니다.

# 프로젝트 설명
사용자가 회원가입 후 로그인을 해서 다른 사용자를 팔로우를 하고, 팔로잉 중인 멤버의 글들을 확인하는 SNS 형태의 프로젝트입니다. k8s 환경에서 인스턴스를 어떻게 구분하는 것이 좋은지에 초점을 맞춰서 설계를 했습니다.<br/>
